Public decks for the **We Will Craft** classes.

## Courses

- **EDP 101**: [Introduction to Electronic Data Processing](https://decks.willcraft.software/edp101/)

## Quick Start

1. Install the dependencies.

    ```bash
    npm install
    ```

2. Start the server.

    ```bash
    npm start
    ```

3. Open http://127.0.0.1:8080/.
